# HybridHUD V9

基于 ESP32-S3 的车载抬头显示系统，通过蓝牙 OBD 连接获取车辆实时数据。

## 硬件平台

- **主控**: ESP32-S3 (Xtensa dual-core 32-bit)
- **显示**: Waveshare ESP32-S3-Touch-AMOLED-1.75 (466×466, 1.75英寸)
- **存储**: PSRAM 8MB + SPI Flash

## 功能特性

### 核心功能
- **蓝牙 OBD 连接**: 自动连接标准蓝牙 OBD-II 适配器
- **实时数据显示**: G值、功率、冷却液温度、电池温度等
- **非阻塞数据同步**: 基于 FreeRTOS 消息队列，UI 渲染与数据采集分离
- **自动重连**: 连接失败后自动重试

### UI 界面
- **Race 页面**: 主显示界面，包含
  - G-Force 环形指示器
  - 实时功率显示（kW）
  - 冷却液温度 / 电池温度
  - 蓝牙状态指示（绿色呼吸=未连接，蓝色闪烁=数据接收中）
- **Bluetooth 页面**: 蓝牙设备扫描与连接

## 软件架构

```
┌─────────────────────────────────────────────────────────┐
│                      Main Task                          │
│                   (UI 渲染 ~60Hz)                        │
└─────────────────────────────────────────────────────────┘
                            ▲
                            │ 消息队列 (非阻塞接收)
                            │
┌─────────────────────────────────────────────────────────┐
│                     OBD Task                            │
│              (数据采集 100ms/2s 轮询)                     │
└─────────────────────────────────────────────────────────┘
                            ▲
                            │
┌─────────────────────────────────────────────────────────┐
│                    BLE Stack                             │
│              (NimBLE + OBD 协议解析)                      │
└─────────────────────────────────────────────────────────┘
                            ▲
                            │
                    ┌───────────────┐
                    │  Bluetooth    │
                    │  OBD Adapter  │
                    └───────────────┘
```

## 数据轮询策略

| 数据类型 | 轮询周期 | 说明 |
|----------|----------|------|
| 功率 (电流/电压) | 100ms | 高频实时 |
| 电池温度 | 2000ms | 中频 |
| 冷却液温度 | ~1-2s | 低频，非关键 |

## 构建与烧录

### 环境要求
- ESP-IDF v5.0+
- Python 3.8+

### 构建步骤

```bash
# 进入项目目录
cd e:\Mygithub\hybridhudv9

# 设置 IDF 环境
idf.py set-target esp32s3

# 配置项目
idf.py menuconfig

# 编译
idf.py build

# 烧录
idf.py -p COM{X} flash monitor
```

### 配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| PSRAM | 启用外部 PSRAM | 8MB Octal |
| LVGL 内存 | 使用 PSRAM 分配 | Custom |
| DMA 缓冲区 | PSRAM 分配 | 8 buffers |

## 消息队列同步机制

```c
// OBD 任务发送数据（非阻塞）
obd_queue_send(&s_data);

// UI 任务接收数据
obd_data_t data;
if (obd_queue_receive(&data)) {
    // 更新 UI
}
```

## 已知问题与限制

- 蓝牙连接需要 OBD 适配器在可视范围内
- 首次连接需要进入 Bluetooth 页面进行配对
- G-Force 数据依赖 IMU 硬件支持
